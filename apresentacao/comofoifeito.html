Quais foram os recursos utilizados?


Utilizamos como editor de código fonte o "Visual Code Studio" que usamos para a leitura de todas as linguagens de programação que serão exibidas a seguir.
Também utilizamos o "HTML" como uma linguagem de marcação, tal uso para o ajuste de todas as caixa de texto, botões, links, etc.
Utilizamos o "CSS", que é um mecanismo para adicionar estilos a uma página web e separa o conteúdo da representação visual do site (semelhante ao HTML).
Para salvar os dados recebidos no nosso projeto (vendedores, vendas, fornecedores, etc.), utilizamos o MySql para o manuseio dessa demanda, o MySql é um banco de dados que vem junto com Xampp.
E para a programação principal, o PHP foi o nosso escolhido, com ele programamos todo o banco de dados (MySql), programamos o sistema do projeto, a função de cada botão, caixa de texto, mensagem, etc (em alguns casos, alteramos até alguns layouts e aparências).
